Class MxDbText

MxDbText 绘制文字

describe

适用于标签标注类型的文字(永远面向相机的平面)

example
 obj = new MxDbText()
obj.rxInit()
obj.position = new THREE.Vector3()
obj.backgroundColor = '#ff0000'
obj.padding = 5
obj.strokeWidth = 3
obj.strokeColor = "#3a8bd2"
obj.borderColor = "#ff00ff"
obj.borderWidth = 2
obj.borderRadius = 2
obj.opacity = 0.8
obj.text  = 'This is\nsome multi-line\ntext'
obj.height = 100000
obj.color = "#fff000"
